Pandas Project

Introduction

This project was part of a unit on using Google Colab along with NumPy. Through Colab, we were to import a large CSV dataset through a method of our choice and convert it to a dataframe. Then, we would run several different methods of analysis through it and create some NumPy graphs to answer a research question. My dataset was a set of the best-selling video games of all time, which I found on the Kaggle website and imported through their API. My research question was to see which game publishers are the most successful. I was able to answer that it was Nintendo!
